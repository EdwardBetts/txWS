====
txWS
====

txWS (prounounced "Twisted WebSockets") is a small, short, simple library for
adding WebSockets support to your favorite Twisted applications.

Versions
========

txWS supports the following versions of the WebSockets draft:

 * Version 76
   * Hixie-76
   * HyBi-00
 * Version 7
   * HyBi-07
